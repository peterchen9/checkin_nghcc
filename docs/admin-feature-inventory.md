# 後台功能盤點

後台參考網址：https://checkin.nghcc.org.tw/admin.html

## 功能清單

- 打卡紀錄查詢：查詢會友、同工、課程、活動或聚會的報到與打卡紀錄。
- 日期區間查詢：依起訖日期篩選紀錄。
- 類別篩選：依聚會、課程、活動、小組、同工打卡等類別篩選。
- Excel 匯出：將查詢結果或管理清單匯出為 Excel。
- 課程與活動管理：維護課程、聚會、活動名稱、日期、狀態與相關設定。
- 員工名單管理：維護同工或員工名單與所屬單位。
- 通訊錄管理：直接使用既有 `192.168.16.240:8001` 通訊錄，透過 `/addressbook/` 代理入口，不修改原內容。
- 報名清單：查詢與管理課程活動報名名單。
- 繳費狀態：紀錄與查詢課程或活動相關繳費狀態。
- 上課狀況：紀錄與查詢課程出席或上課狀態。

## 注意事項

後台涉及管理權限與個資資料，後續設計需納入登入、角色權限、操作紀錄、匯出限制與備份策略。通訊錄若需寫入、同步或資料搬遷，必須另行定義權限與審核流程。
