# 前台功能盤點

前台參考網址：https://checkin.nghcc.org.tw/index.html

## 功能清單

- 聚會類型選擇：選擇主日聚會、成主課程、特別聚會、牧區小組聚會或同工打卡等情境。
- 課程/聚會選擇：依選定類型列出可報到的課程、活動或聚會。
- 牧區/小組選擇：支援牧區與小組聚會報到時的分類選擇。
- 所屬單位選擇：支援同工或名單資料依單位進行篩選與報到。
- QR Code 掃描報到：透過掃描 QR Code 快速完成報到。
- 手動搜尋報到：在無法掃描 QR Code 或需人工查找時，以姓名或資料搜尋完成報到。
- 同工打卡安全碼：同工打卡時需通過安全碼驗證，降低誤打卡或非授權操作風險。

## 注意事項

本文件目前為功能盤點，不代表已完成重建。後續需補充畫面流程、欄位定義、錯誤狀態與 API 對應。
