# 系統總覽

## 系統名稱

教會智能報到系統

## 前台網址與功能摘要

參考網址：https://checkin.nghcc.org.tw/index.html

前台提供會友與同工進行聚會、課程、活動與小組相關報到。主要功能包含主日聚會、成主課程、特別聚會、牧區小組聚會、同工打卡、QR Code 掃描報到、QR Code 30x30mm 標籤列印、手動搜尋報到，以及同工打卡安全碼驗證。

## 後台網址與功能摘要

參考網址：https://checkin.nghcc.org.tw/admin.html

後台提供管理者查詢與管理報到、打卡、課程活動與名單資料。主要功能包含打卡紀錄查詢、Excel 匯出、課程與活動管理、員工名單管理、通訊錄管理、報名清單、繳費狀態與上課狀況。通訊錄目前直接使用既有 192.168.16.240:8001 服務，透過 /addressbook/ 代理入口使用，不修改原內容。

## 使用對象

- 會友：進行聚會、課程、活動與小組報到。
- 同工：進行同工打卡與相關報到作業。
- 管理者：管理課程活動、名單資料、報到紀錄、匯出與查詢。

## 系統邊界

本系統僅處理報到、打卡、課程活動、通訊錄、員工名單與查詢匯出。不得擴張到既有「北門行政作業平台」或其他系統，也不得在目前階段直接連線或修改 production 資料庫。
